## ⚙️ Lingkungan Pengembangan: Laragon

Pada modul ini, saya menggunakan **Laragon** sebagai lingkungan pengembangan lokal untuk menjalankan proyek Laravel.  
Laragon dipilih karena lebih **ringan**, **stabil**, dan **otomatis mendeteksi virtual host**, sehingga sangat mendukung proses pengembangan framework Laravel.
